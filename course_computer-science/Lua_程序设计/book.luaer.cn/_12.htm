-- defines a factorial ________

function fact (n)

    if n == 0 then

       return 1

    else

       return n * fact(n-1)

    end

end

 

print( "enter a number:" )

a = io.read( "*number" )      -- read a number

print(fact(a))

这个例子定义了一个函数，计算输入参数 n 的阶乘；本例要求用户输入一个数字 n ，然后打印 n 的阶乘。
