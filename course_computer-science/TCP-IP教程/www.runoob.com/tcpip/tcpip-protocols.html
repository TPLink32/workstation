TCP/IP 协议

TCP/IP 是不同的通信协议的大集合。

协议族

TCP/IP 是基于 TCP 和 __ 这两个最初的协议之上的不同的通信协议的大集合。

TCP - 传输控制协议

TCP 用于从应用程序到网络的数据传输控制。

TCP 负责在数据传送之前将它们分割为 IP 包，然后在它们到达的时候将它们重组。

IP - 网际协议（Internet Protocol）

IP 负责计算机之间的通信。

IP 负责在因特网上发送和接收数据包。

HTTP - 超文本传输协议(Hyper Text ________ Protocol)

HTTP 负责 web 服务器与 ___ 浏览器之间的通信。

HTTP 用于从 web 客户端（浏览器）向 ___ 服务器发送请求，并从 web 服务器向 web _______（__）。

HTTPS - 安全的 HTTP（Secure ____）

HTTPS 负责在 web 服务器和 ___ 浏览器之间的安全通信。

作为有代表性的应用，HTTPS 会用于处理信用卡交易和其他的敏感数据。

SSL - 安全套接字层（Secure Sockets _____）

SSL 协议用于为安全数据传输加密数据。

SMTP - 简易邮件传输协议（Simple Mail ________ Protocol）

SMTP 用于电子邮件的传输。

MIME - 多用途因特网邮件扩展（Multi-purpose Internet ____ Extensions）

MIME 协议使 SMTP 有能力通过 ___/__ 网络传输多媒体文件，包括声音、视频和二进制数据。

IMAP - 因特网消息访问协议（Internet Message ______ Protocol）

IMAP 用于存储和取回电子邮件。

POP - 邮局协议（Post Office ________）

POP 用于从电子邮件服务器向个人电脑下载电子邮件。

FTP - 文件传输协议（File Transfer ________）

FTP 负责计算机之间的文件传输。

NTP - 网络时间协议（Network Time ________）

NTP 用于在计算机之间同步时间（钟）。

DHCP - 动态主机配置协议（Dynamic Host _____________ Protocol）

DHCP 用于向网络中的计算机分配动态 IP 地址。

SNMP - 简单网络管理协议（Simple Network __________ Protocol）

SNMP 用于计算机网络的管理。

LDAP - 轻量级的目录访问协议（Lightweight Directory ______ Protocol）

LDAP 用于从因特网搜集关于用户和电子邮件地址的信息。

ICMP - 因特网消息控制协议（Internet Control _______ Protocol）

ICMP 负责网络中的错误处理。

ARP - 地址解析协议（Address Resolution ________）

ARP - 用于通过 IP _____ IP 地址的计算机网卡的硬件地址。

RARP - 反向地址转换协议（Reverse Address __________ Protocol）

RARP 用于通过 IP 查找基于硬件地址的计算机网卡的 __ 地址。

BOOTP - 自举协议（Boot Protocol）

BOOTP 用于从网络启动计算机。

PPTP - 点对点隧道协议（Point to _____ Tunneling Protocol）

PPTP 用于私人网络之间的连接（隧道）。
