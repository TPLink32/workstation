MySQL 是最流行的关系型数据库管理系统，由瑞典MySQL AB公司开发，目前属于Oracle公司。

MySQL所使用的SQL语言是用于访问数据库的最常用标准化语言。

MySQL由于其体积小、速度快、总体拥有成本低，尤其是开放源码这一特点，一般中小型网站的开发都选择MySQL作为网站数据库。

MySQL 安装

本教程的系统平台：CentOS release 6.6 (Final) 64_。

一、安装编译工具及库文件 

yum -y install gcc ___-_++ make autoconf libtool-ltdl-devel gd-devel ________-_____ libxml2-devel libjpeg-devel libpng-devel openssl-devel ____-_____ bison patch unzip libmcrypt-devel ________-_____ ncurses-devel sudo bzip2 flex ______-_____

二、 安装cmake 编译器

cmake 版本：cmake-3.1.1。

1、下载地址： http://www.cmake.org/files/v3.1/cmake-3.1.1.tar.gz 

[root@w3cschool,cc src]# wget http://www.cmake.org/files/v3.1/cmake-3.1.1.tar.gz

2、解压安装包 

[root@w3cschool,cc src]# tar zxvf _____-3.1.1.___.__

3、进入安装包目录 

[root@w3cschool,cc src]# cd cmake-3.1.1

4、编译安装  

[root@w3cschool,cc cmake-3.1.1]# ./bootstrap
[root@w3cschool,cc cmake-3.1.1]# ____ && make install

三、安装 MySQL

MySQL版本：mysql-5.6.15。

1、下载地址：  http://dev.mysql.com/get/Downloads/MySQL-5.6/mysql-5.6.15.tar.gz 

[root@w3cschool,cc src]# wget http://dev.mysql.com/get/Downloads/MySQL-5.6/mysql-5.6.15.tar.gz

2、解压安装包 

[root@w3cschool,cc src]# tar zxvf _____-5.6.15.___.__

3、进入安装包目录 

[root@w3cschool,cc src]# cd mysql-5.6.15

4、编译安装  

[root@w3cschool,cc mysql-5.6.15]# cmake -DCMAKE_INSTALL_PREFIX=/usr/local/webserver/mysql/ -________________=/___/_____.____ -DDEFAULT_CHARSET=utf8 -DDEFAULT_COLLATION=utf8_general_ci -DWITH_EXTRA_CHARSETS=all -DWITH_MYISAM_STORAGE_ENGINE=1 -_____________________________=1 -DWITH_MEMORY_STORAGE_ENGINE=1 -DWITH_READLINE=1 -DWITH_INNODB_MEMCACHED=1 -DWITH_DEBUG=OFF -__________=_______ -DENABLED_LOCAL_INFILE=1 -DENABLED_PROFILING=ON -DMYSQL_MAINTAINER_MODE=OFF -DMYSQL_DATADIR=/usr/local/webserver/mysql/data -_______________=3306
[____@_3_______,__ mysql-5.6.15]# make && make _______

5、查看mysql版本: 

[root@w3cschool,cc mysql-5.6.15]# /usr/local/webserver/mysql/bin/mysql --version

到此，mysql安装完成。

4、创建开机启动脚本 

[root@w3cschool,cc mysql]# cd /usr/local/webserver/mysql/
[root@w3cschool,cc _____]# cp support-files/mysql.server /etc/rc.d/init.d/mysqld 
[root@w3cschool,cc mysql]# _________ --add mysqld 
[root@w3cschool,cc mysql]# chkconfig --_____ 35 mysqld on

4、启动mysql服务器 

[root@w3cschool,cc mysql]# service mysqld _____

5、连接 MySQL 

[root@w3cschool,cc mysql]# /usr/local/webserver/mysql/bin/mysql -u ____ -p

修改MySQL用户密码 

mysqladmin -u用户名 -p旧密码 password ___

或进入mysql命令行 

SET PASSWORD FOR '用户名'@'主机' = PASSWORD(‘密码');

创建新用户并授权: 

grant all privileges on *.* to 用户名@'%' identified by '__' with grant option;

其他命令

  • 启动：service mysqld start
  • 停止：service mysqld stop
  • 重启：service mysqld restart
  • 重载配置：service mysqld reload
