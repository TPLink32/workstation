Redis QUIT命令

由 十月的星空 翻译整理   注： 由于水平有限，欢迎指正。转载请务必注明出处。
 上一篇 下一篇  

Redis QUIT命令要求服务器关闭连接。该连接只要所有未完成操作写入到客户端后就关闭。

返回值

返回字符串：OK

语法

Redis QUIT命令基本语法如下所示： 

 redis 127.0.0.1:6379> QUIT

例子 

 redis 127.0.0.1:6379> QUIT
