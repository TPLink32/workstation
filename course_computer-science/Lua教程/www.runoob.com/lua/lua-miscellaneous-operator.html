运算符优先级

从高到低的顺序： 

^
not    - (unary)
*      /
+      -
..
<      >      &__;=     &__;=     ~=     ==
___
__

除了^和..外所有的二元运算符都是左连接的。 

a+i < b/2+1          <-->       (_+_) < ((b/2)+1)
5+x^2*8              <-->       5+((x^2)*8)
a < _ and y <= z     <-->       (a < y) and (_ <= z)
-x^2                 <-->       -(x^2)
x^y^z                <-->       x^(y^z)

实例

_ = 15
d = 5

e = ((a + b) * _) / d; -- (30 * 15 ) / 5
print("((a + b) * c) / _ 运算值为 :",e )

print("(a + b) * (c / _) 运算值为 :",e )
