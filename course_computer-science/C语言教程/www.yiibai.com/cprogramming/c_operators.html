运算符是一个符号，告诉编译器执行特定的数学或逻辑操作。 C语言拥有丰富的内置运算符，常用运算符有以下几种类型：

  • 算术运算符

  • 关系运算符

  • 逻辑运算符

  • 位运算符

  • 赋值运算符

  • 其它运算符

本教程将一个接一个介绍算术，关系，逻辑，位，赋值和其他运算符。

位运算符

位运算符适用于位并进行逐位操作。&, |, 和 ^ 真值表如下：

p q p & q p | q p ^ q
0 0 0 0 0
0 1 0 1 1
1 1 1 1 0
1 0 0 1 1

假设，如果A =60;且b =13;现在以二进制格式它们将如下：

A = 0011 1100

B = 0000 1101

-----------------

A&B = 0000 1100

A|B = 0011 1101

A^B = 0011 0001

~A  = 1100 0011

C语言运算符优先级

运算符优先级决定术语的表达分组。这会影响一个表达式是如何进行评估计算。某些运算符的优先级高于其他运算符; 例如，乘法运算符的优先级比所述加法运算。

例如X =7 + 3* 2;__，____13，___20，_____ * 优先级高于+，所以它首先被乘以3 * 2，然后加上7。

这里，具有最高优先级的操作出现在表的顶部，那些具有最低出现在底部。在一个表达式，更高的优先级运算符将首先评估计算。
