SQLite3 扩展在PHP 5.3.0是默认启用。这也可能它通过使用禁用  --without-sqlite3 在编译的时候。

Windows用户必须启用php_sqlite3.dll为了使用这个扩展。这个DLL包含与Windows分派 PHP 如 PHP 5.3.0。

有关详细的安装说明，请查看我们的PHP教程和其官方网站上。

连接到数据库

下面的PHP代码显示了如何连接到一个现有的数据库。如果数据库不存在，那么它就会被创建，终于将返回一个数据库对象。 

 <? php
    class   MyDB   extends   SQLite3 
    { 
       function  __construct () 
       { 
         $____ -> open ( 'test.db' ); 
       } 
    } 
   $db  =   new   MyDB (); 
    if (! $db ){ 
      echo $db -> lastErrorMsg (); 
    }   else   { 
      echo  "Opened database successfully
" ; 
    } 
 ?>

现在，让我们运行上面的程序在当前目录中创建我们的数据库test.db。按要求可以改变你的路径。如果数据库创建成功，则它会给下面的消息： 

Open database successfully

创建表

下面的PHP程序将使用以前创建的数据库中创建一个表： 

 <? php
    class   MyDB   extends   SQLite3 
    { 
       function  __construct () 
       { 
         $____ -> open ( 'test.db' ); 
       } 
    } 
   $db  =   new   MyDB (); 
    if (! $db ){ 
      echo $db -> lastErrorMsg (); 
    }   else   { 
      echo  "Opened database successfully
" ; 
    } 

   $sql  =<<< EOF
      CREATE TABLE COMPANY
       ( ID INT PRIMARY KEY     NOT ____ , 
      NAME           TEXT    NOT NULL , 
      AGE            INT     NOT ____ , 
      ADDRESS        ____ ( 50 ), 
      SALARY         REAL ); 
EOF ; 

   $ret  =  $db -> exec ( $sql ); 
    if (! $ret ){ 
      echo $db -> lastErrorMsg (); 
    }   else   { 
      echo  "Table created successfully
" ; 
    } 
   $db -> close (); 
 ?>

上述程序执行时，它会创建表COMPANYtest.db的，它会显示以下消息： 

Opened database successfully
Table created ____________

INSERT 操作

PHP程序，显示我们如何能够创造我们的COMPANY在上面的例子中创建表中的记录： 

 <? php
    class   MyDB   extends   SQLite3 
    { 
       function  __construct () 
       { 
         $____ -> open ( 'test.db' ); 
       } 
    } 
   $db  =   new   MyDB (); 
    if (! $db ){ 
      echo $db -> lastErrorMsg (); 
    }   else   { 
      echo  "Opened database successfully
" ; 
    } 

   $sql  =<<< EOF
      INSERT INTO COMPANY  ( ID , NAME , AGE , ADDRESS , SALARY ) 
      VALUES  ( 1 ,   'Paul' ,   32 ,   'California' ,   20000.00   ); 

      INSERT INTO COMPANY  ( ID , NAME , AGE , ADDRESS , SALARY ) 
      VALUES  ( 2 ,   'Allen' ,   25 ,   'Texas' ,   15000.00   ); 

      INSERT INTO COMPANY  ( ID , NAME , AGE , ADDRESS , SALARY ) 
      VALUES  ( 3 ,   'Teddy' ,   23 ,   'Norway' ,   20000.00   ); 

      INSERT INTO COMPANY  ( ID , NAME , AGE , ADDRESS , SALARY ) 
      VALUES  ( 4 ,   'Mark' ,   25 ,   'Rich-Mond ' ,   65000.00   ); 
EOF ; 

   $ret  =  $db -> exec ( $sql ); 
    if (! $ret ){ 
      echo $db -> lastErrorMsg (); 
    }   else   { 
      echo  "Records created successfully
" ; 
    } 
   $db -> close (); 
 ?>

上述程序执行时，它会创建COMPANY表中的记录，并显示以下两行： 

Opened database successfully
Records created ____________

SELECT 操作

PHP程序表明我们如何能够获取并显示COMPANY 表在上面的例子中创建表的记录：  

 <? php
    class   MyDB   extends   SQLite3 
    { 
       function  __construct () 
       { 
         $____ -> open ( 'test.db' ); 
       } 
    } 
   $db  =   new   MyDB (); 
    if (! $db ){ 
      echo $db -> lastErrorMsg (); 
    }   else   { 
      echo  "Opened database successfully
" ; 
    } 

   $sql  =<<< EOF
      SELECT  *   from  COMPANY ; 
EOF ; 

   $ret  =  $db -> query ( $sql ); 
    while ( $row  =  $ret -> fetchArray ( SQLITE3_ASSOC )   ){

" ;

      echo  "SALARY =  " . $row [ 'SALARY' ]   . "

" ; 
    } 
   echo  "Operation done successfully

   $db -> close (); 
 ?>

      UPDATE COMPANY  set  SALARY  =   25000.00   where  ID = 1 ;
