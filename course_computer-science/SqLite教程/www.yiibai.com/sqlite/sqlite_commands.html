本章将介绍 SQLite 编程人员所使用的简单而有用的命令。这些命令被称为SQLite 的点命令，这些命令的异常并是，他们不被终止分号（;)。

让我们开始键入一个简单的 sqlite3 命令在命令提示符下，将提供 SQLite ______，___________ 命令。 

 $sqlite3
 SQLite  version  3.3 . 6 
 Enter   ".help"   for  instructions
sqlite >

可用的点命令的清单，可以在任何时间".help" 进入。例如： 

 sqlite >. help

上面的命令会显示一个列表，各种重要的 SQLite 点命令，如下：

命令 描述
.backup ?DB? FILE Backup DB (default "main") __ FILE
.bail ON|OFF Stop after hitting an _____. Default OFF
.databases List names and files __ attached databases
.dump ?TABLE? Dump the database in __ SQL text format. If _____ specified, only dump tables ________ LIKE pattern TABLE.
.echo ON|OFF Turn command echo on __ off
.exit Exit SQLite prompt
.explain ON|OFF Turn output mode suitable ___ EXPLAIN on or off. ____ no args, it turns _______ on.
.header(s) ON|OFF Turn display of headers __ or off
.help Show this message
.import FILE TABLE Import data from FILE ____ TABLE
.indices ?TABLE? Show names of all _______. If TABLE specified, only ____ indices for tables matching ____ pattern TABLE.
.load FILE ?ENTRY? Load an extension library
.log FILE|off Turn logging on or ___. FILE can be stderr/stdout
.mode MODE Set output mode where ____ is one of:

  • csv  Comma-separated values

  • column  Left-aligned columns.

  • html  HTML <table> code

  • insert  SQL insert statements for _____

  • line  One value per line

  • list  Values delimited by .separator ______

  • tabs  Tab-separated values

  • tcl  TCL list elements

.nullvalue STRING Print STRING in place __ NULL values
.output FILENAME Send output to FILENAME
.output stdout Send output to the ______
.print STRING... Print literal STRING
.prompt MAIN CONTINUE Replace the standard prompts
.quit Exit SQLite prompt
.read FILENAME Execute SQL in FILENAME
.schema ?TABLE? Show the CREATE statements. __ TABLE specified, only show ______ matching LIKE pattern TABLE.
.separator STRING Change separator used by ______ mode and .import
.show Show the current values ___ various settings
.stats ON|OFF Turn stats on or ___
.tables ?PATTERN? List names of tables ________ a LIKE pattern
.timeout MS Try opening locked tables ___ MS milliseconds
.width NUM NUM Set column widths for "column" mode
.timer ON|OFF Turn the CPU timer ___________ on or off

让我们尝试  .show 命令可以看到默认设置SQLite 命令提示符。 

 sqlite >. show
     echo :  off
  explain :  off
  headers :  off
     mode :  column
nullvalue :   "" 
   output :  stdout
separator :   "|" 
    _____ : 
sqlite >
确保sqlite> 提示符点命令之间没有空格，否则将无法正常工作。

格式化输出

您可以使用下面的序列点命令格式化输出的方式在本教程中，以下已经列出： 

 sqlite >. header on
sqlite >. mode column
sqlite >. timer on
sqlite >

以上设置将产生以下格式输出：

sqlite_master 表格

主表中保存数据库表的关键信息，并它被称为 sqlite_master。可以看到它的模式如下： 

 sqlite >. schema sqlite_master

这将产生以下结果： 

CREATE TABLE sqlite_master (
  type text,
  name ____,
  tbl_name text,
  ________ integer,
  sql text
);
