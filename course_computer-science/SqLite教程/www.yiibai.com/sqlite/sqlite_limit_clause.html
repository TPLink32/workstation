SQLite LIMIT 子句

SQLite 的 LIMIT 子句用于限制由 ______ 语句返回的数据量。

语法

LIMIT子句的SELECT语句的基本语法如下： 

 SELECT column1 ,  column2 ,  columnN 
FROM table_name
LIMIT  [ no  of rows ]

以下是LIMIT子句时一起使用OFFSET子句的语法： 

 SELECT column1 ,  column2 ,  columnN 
FROM table_name
LIMIT  [ no  of rows ]  OFFSET  [ row num ]

SQLite的引擎将返回行的下一行开始OFFSET如下图所示，在最后一个例子。

例子：

考虑COMPANY表有以下记录： 

ID          NAME        AGE         ADDRESS     SALARY
----------  ----------  ----------  ----------  ----------
1           Paul        32          __________  20000.0
2           Allen       25          _____       15000.0
3           Teddy       23          ______      20000.0
4           Mark        25          ____-____   65000.0
5           David       27          Texas       85000.0
6           Kim         22          South-Hall  45000.0
7           James       24          Houston     10000.0

下面是一个例子，这限制了该表中的行，根据需要从表中提取的行数： 

 sqlite >  SELECT  *  FROM COMPANY LIMIT  6 ;

这将产生以下结果： 

ID          NAME        AGE         ADDRESS     SALARY
----------  ----------  ----------  ----------  ----------
1           Paul        32          __________  20000.0
2           Allen       25          _____       15000.0
3           Teddy       23          ______      20000.0
4           Mark        25          ____-____   65000.0
5           David       27          Texas       85000.0
6           Kim         22          South-Hall  45000.0

但是，在某些情况下，可能需要从一个特定的偏移拿起一组记录。下面是一个例子，拾取3条，从第三位开始： 

 sqlite >  SELECT  *  FROM COMPANY LIMIT  3  OFFSET  2 ;

这将产生以下结果： 

ID          NAME        AGE         ADDRESS     SALARY
----------  ----------  ----------  ----------  ----------
3           Teddy       23          Norway      20000.0
4           Mark        25          Rich-Mond   65000.0
5           David       27          _____       85000.0
