C标准库参考

C语言是一种通用的，程序性命令式的计算机编程语言，由Dennis M.里奇于1972年在贝尔电话实验室开发的Unix操作系统语言。

C语言是应用最广泛的计算机语言，它与Java编程语言保持排名第一，都普遍受欢迎，是现代软件编程人员广泛使用的计算机编程语言。

C标准库是一套C的内置函数，常量和头文件一样<stdio.h>，<stdlib.h>，<math.h>等这个库将作为C程序员参考手册。

读者

C标准库是C程序员，帮助他们在其涉及到系统编程项目时作为参考。所有的C函数已经在易于理解和解释，它们可以复制并粘贴到C项目中。

前提条件

对C++编程语言有基本的了解能帮助理解包含在本库中的C内置功能。
