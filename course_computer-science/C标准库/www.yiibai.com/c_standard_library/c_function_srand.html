C库函数void srand(unsigned int seed) 种子rand函数所使用的随机数发生器。

声明

下面是函数 srand() 函数的声明。 

 void  srand ( unsigned   int  seed )

参数

  • seed -- 这是一个整数的值被用来作为种子由伪随机数生成器算法。

返回值

这个函数不返回任何值。

例子

下面的例子演示了如何使用 srand() 函数功能。 

 #include   <stdio.h> 
 #include   <stdlib.h> 
 #include   <time.h> 

 int  main () 
 { 
    int  i ,  n ; 
    time_t  t ; 
   
   n  =   5 ; 
   
    /* Intializes random number _________ */ 
   srand (( unsigned )  time (& t )); 

    /* Print 5 random _______ from 0 to 50 */ 
    for (  i  =   0   ;  i  <  n  ;  i ++   )   { 
      printf ( "%d
" ,  rand ()   %   50 ); 
    } 
   
   return ( 0 ); 
 }

让我们编译和运行上面的程序，这将产生以下结果： 

38
45
29
29
47
