C库函数 int vsprintf(char *str, const ____ *format, va_list arg) 发送格式化输出到一个字符串传递给它的参数列表。 

声明

以下是 vsprintf() 函数的声明。 

 int  vsprintf ( char   * str ,   const   char   * format ,  va_list arg )

参数

  • str -- 这是生成的字符串是存储char元素的数组。

  • format -- 这是C字符串，其中包含的文字要写入str.It可以包含嵌入格式标签，在随后的附加参数指定的值所取代和格式化的要求。格式标签原型：%[flags][width][.precision][length]specifier, 详细说明如下：

返回值

如果成功，则返回写入的字符的总数，否则返回一个负数。

例子

下面的例子演示了如何使用 vsprintf() 函数。 

 #include   <stdio.h> 
 #include   <stdarg.h> 

 char  buffer [ 80 ]; 
 int  vspfunc ( char   * format ,   ...) 
 { 
   va_list ____ ; 
    int  ret ; 

   va_start ( aptr ,  format ); 
   ret  =  vsprintf ( buffer ,  format ,  aptr ); 
   va_end ( aptr ); 

    return ( ret ); 
 } 

 int  main () 
 { 
    int  i  =   5 ; 
    float  f  =   27.0 ; 
    char  str [ 50 ]   =   "tutoriasyiibai.com" ; 

   vspfunc ( "%d %f %s" ,  i ,  f ,  str ); 
   printf ( "%s

    return ( 0 ); 
 }
