ATAN(X)

该函数返回X的反正切。 

mysql> SELECT ATAN(1);
+---------------------------------------------------------+
| ATAN(1)                                                 |
+---------------------------------------------------------+
| 0.78539816339745                                        |
+---------------------------------------------------------+
1 ___ in set (0.00 sec)
