Here are links to some Lua graphics software that don't fit to LibrariesAndBindings but is free[1] and can be interesting.

  • 2D
    • [Aroma] (5.1) - 2D game engine for Google Chrome Native Client. Implements parts of the LÖVE Lua API to ease porting. (MIT license)
    • [blitwizard] (5.2) - a cross-platform 2D engine with easy networking, sound, physics. Completely scriptable with Lua. (zlib license)
    • [CEGUI] (5.0/5.1) - a cross-platform library that provides windowing & widgets for graphics APIs/engines where such functionality is not natively available or severely lacking. Includes renderers for OpenGL, DirectX, Irrlicht, OGRE, Crystal Space, OpenSceneGraph; provides interfaces to write your own renderers. Uses Lua for scripting. (MIT license)
    • [info-beamer] (5.1) - live coding presentation framework for Linux. Completely scriptable with Lua. (BSD license)
    • [Gideros] (5.1) - 2D game engine for iOS and Android. Completely scriptable with Lua. (MIT license)
    • [LÖVE] (5.1) - a cross-platform 2D game engine. Completely scriptable with Lua. (zlib license)
    • [Moai SDK] (5.1) - 2D game engine for iOS and Android, Mac OS X, Windows and Google Chrome (Native Client). Completely scriptable with Lua. (CPAL[2])
    • [YAE] [3] (5.3) - a cross-platform 2D game engine. Works on Windows, Mac OS X, Linux, Android and iOS. Completely scriptable with Lua and MoonScript?. (MIT license)
    • [Scrupp] (5.1) - a cross-platform 2D game engine. Completely scriptable with Lua. (MIT license)
    • [T-Engine4] (5.1) - a cross-platform 2D [roguelike] game engine. Supports both "old school" ASCII display and OpenGL/SDL graphics. Completely scriptable with Lua. (GPL)
    • [Verge] (5.?) - a cross-platform 2D game engine. Completely scriptable with Lua. (BSD license)
  • 3D
    • [Aleph One] [4] (5.0) - a cross-platform 3D [FPS] game engine. Uses Lua for in-game scripting (such as scenarios, mods and total conversions)[5]. (GPL)
    • [Apocalyx] (5.1) - 3D engine for Windows/OpenGL. Completely scriptable with Lua. (GPL)
    • [Cafu (Ca3DE)] [6] (5.1) - a cross-platform 3D graphics engine and game development kit. Uses Lua for GUI and entity (map) scripting. (GPL)
    • [Electro] [7] [8] (5.1) - a cross-platform 3D visualisation engine. Based on the MPI[9] process model and designed for use on both cluster-driven tiled displays and desktop systems. Completely scriptable with Lua. (GPL)
    • [LuaAV] (5.1) - a cross-platform 3D visualisation and real-time audio synthesis engine. Completely scriptable with Lua. (BSD-like license)
    • [Luxinia] (5.1) - 3D engine for Windows/OpenGL. Completely scriptable with Lua. (MIT/BSD-like license)
    • [Maratis] (5.1) - a cross-platform 3D game engine and visual environment. Scriptable with Lua. (GPL)
    • [Nebula Device] [10] (5.0) - a 3D game engine for Windows/DirectX. Scriptable with Lua, Python, Ruby, Tcl. (MIT license)
    • [Spring] [11] (5.1) - a cross-platform 3D [RTS] game engine. Completely scriptable with Lua[12]. (GPL)
    • [Zombie] (5.0) - a successor of Nebula 2. (BSD license)
