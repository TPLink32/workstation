There are two chat rooms for discussing Lua:

  • [irc.freenode.net #lua]: Ask all kinds of questions.
    • There are a number of bots in the channel that can execute Lua code and respond with the results, these include YetAnotherLuaBot (yalb), LuaBot and BatBot. For larger or longer experimentation with Lua, try the [live demo], [codepad] or [repl.it].
    • If you need a channel operator for any reason, look for ventonegro, rjek, deryni.
    • Be aware that the channel, like any other, may be logged by specific users. [1]
    • Everyone is welcome! The channel has a social nature so don't be surprised if people are talking about other things to fill the gaps.
  • [irc.freenode.net #lua-support]:
    • Dedicated to handling support based questions for novice and pro alike
    • NOTICE: This channel is publicly logged: http://wy.si/1b
    • Be aware that off-topic is not tolerated, you will get banned (read more: https://freenode.net/policy.shtml#offtopic)
    • Current channel operator: develCuy

You'll need a [client] to join the rooms, such as [freenode's web client].

Here are some guidelines for the channels:

  • Try to remain on-topic.

  • Read the [Lua manual], the [book], and the LuaFaq before asking questions.
  • Learn [to ask a question] and [how to ask questions the smart way].
  • You will get more answers if you ask in English, but there are usually speakers of Spanish and Portuguese, and often of other languages.
  • Learn [how to report a bug].

  • Don't ask if you can ask a question — just ask it.
  • Don't ask if anyone can help — just ask.
  • Don't ask if anyone knows about something — just ask.

  • Don't private message.
  • Don't post code to IRC; use pastebins like: http://lua.pastey.net/ http://lua.pastebin.com/ http://codepad.org

In short, learn [how to get useful help on IRC].
