This page contains a list of the author's (recent and dated) contributions to Lua.

Recent Contributions

  • [axe.lua] - a Lua API interface for [BlitzMAX] (available and stable)
  • BlitzLua - a Lua run-time environment written in [BlitzMAX] (pre-alpha version released, still under development)

Miscellany

  • IntegerDomain - a short Lua script to calculate the largest integer number supported by Lua
  • LuaLogo - the Lua-Logo in PNG format (various sizes, with alpha channel)
  • [Lua.icns] - Lua icons file for MacOS X

Please note: the Lua logos are Graphic design by A. Nakonechnyj. Copyright © 1998. All rights reserved. (see http://www.lua.org/images)

Dated Contributions

  • [Lua_5.0.syn] - a Lua 5.0 compliant syntax definition file for the [TextPad] editor (Win32 only)

  • [Win32 distributions] of Lua 4.0 (including bugfixes published on June 11, 2002), TkLua? 4.0a1 and LuaSocket? 1.4 (combined into a single executable for stand-alone Lua scripts)
  • [initial programming examples] for Lua newbies (like me)
