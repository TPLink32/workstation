Simple Command Line Script Execution
wiki

To Execute a script from the command line, e.g from a NSLU2 running openWrt Kamikaze 8.09rc2 with Lua in the path:

echo "print(\"Hello, World\")" > test.lua

lua test.lua

The result is:

Hello, World
