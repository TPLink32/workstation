Scite Scripts
wiki

This is a place for sharing your small Lua scripts for the [SciTE] text editor. New pages should generally start with "Scite" so we don't clutter the whole wiki. Note: SciTE 1.74 and later uses Lua 5.1, and as such, scripts that use Lua 5.1-incompatible code may break.

General

  • SciteProgrammers - set up scite as an IDE for C, C++, Java, more, with tool tips and all (in progress).
  • UsingLuaWithScite - a small howto (we should describe how to set lua functions as commands, etc...)
  • SciteUsingUnicode - sample code and functions on using a UTF-8 buffer

Helpers

Little "convenience" scripts like auto completion
  • SciteLatex - 3 useful scripts for writing Latex.
  • SciteMiscScripts - toggle binary values, increment/decrement number, transpose two characters, insert current data, Lua calculator/expression evaluator
  • SciteWordSubstitution - auto-replacement of words in the editor
  • SciteAutoExpansion - automatic expansion of some c keywords such as "if" and "for"
  • SciteWordSelect - select the word under cursor
  • SciteMacroExpander - expanding macros in-line
  • SciteTextFolding - outline mode for text documents (like Emacs)
  • SciteManPages - Open a man page about selected word in Scite.
  • SciteOpenPhpLocalhost - Open a PHP file in localhost (instead of try to open the file outside of a web environment, that is not useful)
  • SciteOpenUrl - open selection or text under cursor in default browser
  • SciteOpenFilename - replacement for Ctrl+Shift+O with ability to search for some library and system header files
  • SciteOpenToLine - open and goto line# (compilers produce output with line# like ../foo/../bar.c:1505:27)
  • SciteProcessString - finds and performs operation on strings in a C source file
  • SciteListAllOccurances - finds all occurrences of the selected text in a file
  • SciteMarkWord - marks all occurrences of the word under the cursor or selected text in the current file
  • SciteXmlAutocompletion - automatically closes tags and quotes attributes in XHTML and any XML files
  • SciteCustomFolding - customizable folding and unfolding script
  • SciteDisplayFunctions - finds functions in a PHP source file and displays in UserList?
  • SciteJavadocComment - expands comments to Javadoc-style comments (like in Eclipse)
  • SciteProgrammingUtils - a few simple scripts for C programmers from a lua newbie. Work in Progress!
  • SciteBackupFile - use a script to keep a backup of the file you are editing
  • [ScitePHPSuit] - Optimize SciTe? For PHP
  • SciteAutoCompleteAnyLanguage - Provide auto-completion in any file based purely on the file's contents.
  • SciteFileBrowser - Let to view files in a current directory and open ones.
  • SciteExternalFileBrowser - open external file browser like 'explorer'.
  • SciteInsertDate - insert the date and time at current position
  • SciteEditWithVim - edit current editing file with an external editor such as Vim
  • SciteLineBreak - toggles automatic "line breaking" at given width while typing text

Text Manipulation

  • SciteTabsToSpacesObserveTabstop
  • SciteCleanDocWhitespace - strip trailing spaces and fix indentation
  • SciteSimpleTemplate - apply tabular values to a template text
  • SciteHexify - convert the selection to hex escaped form
  • SciteCleanWhiteLines - strip white lines of a text
  • SciteQuickStartXhtml - a XHTML Quick Start
  • SciteCommentBox - reformats and adds a comment box around a text selection, customizable
  • SciteTitleCase - changes selected text into titlecase
  • SciteHtmlEntities - functions to deal with HTML entities
  • SciteSortSelection - sort a selection of text
  • [Allman-indentation for Java to put to Scites Usermenu]
  • SciteIndentation - reindent selected Lua code
  • SciteDeleteBlankLines - delete all blank lines in a document
  • SciteIndent - indent HTML, PHP, C, CPP, Javascript, embedded Javascript, XML
  • SciteMergeOnChange - keep text synchronized with changes made outside of SciTE.
  • SciteStripTrailings - Strip trailing whitespace on file saving
  • SciteConvertDecHex - converts the selected text to decimal or hex
  • SciteUnicodeInput - replaces the unicode hexadecimal codepoint before cursor, with its UTF-8 encoding

Custom Lexers

  • [Scintillua] supports LPeg lexers ([examples])
  • Recent versions of SciTE have experimental support for writing lexers in Lua: [info]

Syntax highlighting/AST analysis

  • LuaInspect provides a plugin that syntax highlights using intelligent AST analysis (via MetaLua libraries)

Navigation

  • SciteBufferSwitch - accessing recently-used buffers with a drop-down list
  • SciteFuncList - shows the function list of the current editing source file in the output pane
  • SciteTags - support for ctags
  • SciteReadTags - a derivative of SciteTags with a ctags based api generator
  • SciteNumberedBookmarks - Borland-style numbered bookmarks (remember ctrl+K - 1?)

Utilities

  • SciteOther - controlling another instance of SciTE, access menu commands in any instance, and a quiet replacement for os.execute
  • SciteExtMan - a Lua script manager for SciTE
  • SciteFavs - open favourite files easily
  • SciteHexEdit - a basic hex editor. Browsing, editing, saving, with a hex pane and an ASCII pane.
  • SciteAsciiTable - view the editor's 8-bit character set.
  • SciteInplaceCalculator - script to execute an existing formula (uses Lua's loadstring)
  • SciteRunOneScript - allows the user to select a script from a list and run it (uses SciteExtMan)
  • SciteColouriseDemo - primitive styling of a specific buffer using the Lua extension
  • SciteMakeMonospace - forces a buffer to monospace mode using Lua
  • SciteCalculator - a calculator in a buffer, works and looks like a real calculator drawn in text mode
  • SciteLuaPrompt - a simple prompt for using SciTE Lua interactively.
  • SciteDebug - GDB,Lua and Python debugging integrated with SciTE.
  • SciteScriptManager - another Lua script manager for SciTE

File Processing

  • [Lua Exporters] - a set of exporters that works the same as SciTE's normal exporters, plus an enhanced PDF exporter (with line wrapping and kerning), a new SXW ([OpenOffice.org] writer) exporter, a new ABW ([AbiWord 2.2]) exporter, and a new ODT (Open Document) exporter.

Recreation

  • SciteTicTacToe - an improved tic tac toe that acts like a self-contained Lua application.
  • SciteElizaClassic - Joseph Weizenbaum's classic Eliza for SciTE

Others

  • SciteLuaDll - Creating fully functional custom Dlls for Lua
  • [mkottman/scite] - modified SciTE editor including more Lua related additions (Linux and Windows)
  • [scite-for-lua] - SciTE bundle with extensive support for Lua programming (Windows only)
