Binaries and native builds

  • [RiscLua] (5.3.1) - RiscLua 6.00 by GavinWraith; also [1].

The arrival of dynamic loadability, with GCC 4.7.4, to RISC OS means that RiscLua can now revert to being more similar to standard Lua. The riscos library, which provides access to the operating system, is no longer statically linked. Apart from the insertion of requisite require statements to load in libraries that previously were statically linked, nothing need be added to programs that ran for RiscLua 5.xx to make them run for version 6.00.
